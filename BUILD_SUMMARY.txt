                       GETTING STARTED
================================================================================

1. INSTALL DEPENDENCIES
   $ cd /sessions/kind-keen-mayer/heywren-app
   $ npm install

2. CONFIGURE ENVIRONMENT
   $ cp .env.example .env.local
